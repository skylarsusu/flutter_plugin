# flutter_plugin

自定义小部件集合.
包括动画按钮、自定义bottomTabBar、bubbleBottomBar、瀑布流、下拉抽屉、闪屏、自定义卡片、各种动画效果等等
自我记录小效果

## Getting Started

This project is a starting point for a Flutter

Android and/or iOS.
